# 個人 agent 配置

主要用來收集 skills 和同步環境配置使用

## 當前使用的 skills

- `executing-plans`: 引用自 `superpowers`，用於執行複雜計劃。
- `golang-pro`: 引用自 `jeffallan-skills`，增強 Go 語言開發能力。
- `writing-plans`: 引用自 `superpowers`，用於編寫實施計劃。
- `writing-skills`: 引用自 `superpowers`，用於創建和優化技能。

## 致謝與來源

本配置使用了以下優秀的開源技能庫（以 Git Submodule 形式引入）：

- [anthropics/skills](https://github.com/anthropics/skills)
- [google-labs-code/stitch-skills](https://github.com/google-labs-code/stitch-skills)
- [Jeffallan/claude-skills](https://github.com/Jeffallan/claude-skills)
- [obra/superpowers](https://github.com/obra/superpowers)
